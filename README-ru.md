# Проект по изучению программирования на языке ассемблера
[Read in English][en]

## Пример сборки и запуска инструкции
Сборка:
```
$ make source=hello path=./instructions
```

Запуск:
```
$ ./hello/hello.asm-exe
```

Результат:
```
Hello, World!
```
